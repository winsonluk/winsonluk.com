Maglev is a free website template

Built with HTML, CSS and jQuery. Do whatever you like with it.

About

With clean, semantic markup you can hit the ground running and start making Maglev your own immediately.

  • HTML 5

    Clean, semantic markup which is up to modern web standards.

  • CSS 3

    Utilises the power of CSS 3 media queries to provide a robust, responsive website.

  • jQuery

    Entirely optional, but a few little bits of animation around the page are handled by jQuery.
